# PLD Spec Rules

- This directory holds **implemented and verified** general PLD definitions and governance specs only.
- `operating-rules.md`, `guardrails.md`, and `communication.md` are repo-level definitions for long-lived collaboration semantics and boundaries.
- `plugins/parallel-lane-dev/spec/PLD/` must not contain per-execution process docs, scoreboards, or runtime scripts; keep those runtime artefacts under the plugin root `plugins/parallel-lane-dev/` (this repo) or under a consumer repo root `PLD/` tree.
